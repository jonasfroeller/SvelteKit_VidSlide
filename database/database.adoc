= What does the database need to be able to store?

== Brainstorming:

*User:* 

* UID: int 
* Username: string
* Password: string (hashed in api before saving)
* DateTimeCreated: datetime
* LastUpdate: datetime
* ProfilePicture: string (path to image)
* ProfileDescription: string

*Video:*

* VID: int 
* UID: int
* Title: string
* Description: string
* Location: string
* DateTimePosted: datetime
* Views: int
* Shares: int

---

*Feedback:*

* FID: int 
* VID: int 
* Type: string (CHECK Type IN ('positive', 'negative'))

*Comment:*

* CID: int 
* VID: int 
* DateTimePosted: datetime
* TextContent: string
* User: user 
* Parent: comment (replies)

*Following:*

* follower: int
* following: int

*Social:*

* Instagram: string 
* YouTube: string 
* Twitter: string 
* Tiktok: string 
* Custom: website 

*HashTag:*

* HID: int 
* Name: string
